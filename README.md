# 📚 Smart Study Planner (CLI App)

A simple and beginner-friendly **Command Line Interface (CLI)** based Smart Study Planner built using Python.

---

## 🚀 Features

* ✅ Add tasks
* 📋 View all tasks
* ✔ Mark tasks as done
* ❌ Delete tasks
* ⏰ Task duration tracking (in hours)
* ⚠ Reminder for pending tasks

---

## 🛠 Tech Used

* Python (Core concepts)
* Lists & Dictionaries
* Loops and Conditional Statements

---

## ▶ How to Run

1. Clone the repository or download the file
2. Open terminal / command prompt
3. Navigate to project folder
4. Run the program:

```bash
python planner.py
```

---

## 💡 Example Usage

```
----- Smart Study Planner -----

1. Add Task
2. View Task
3. Mark as Done
4. Delete Task
5. Exit
```
